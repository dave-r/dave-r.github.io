Connecting to the Server

First, send us your XBox Gamertag and we'll add you to the server. Once you have been added, follow these steps to connect :
Sign in to your XBox Account
Tap the Play button
Tap the Servers tab.
  • Note : If you see a message about not being able to join servers or realms because of the way your account is set up, you may need to adjust your account privacy settings.
Tap the Add Server button
Enter the server details then tap the Save button
  • Server Name : sonlandia (you can actually use any name you want)
  • Server Address : 34.224.44.86
  • Port : 19132
Back on the Servers tab, scroll to the bottom and you should see the new server. Tap the server button to connect.
